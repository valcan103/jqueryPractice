Carousel gallery
slick.js, jquery, css를 이용하여
자동으로 이미지가 슬라이드 되고,
커서를 올리면 확대되는 효과를 주었습니다.
Hover Effect
css의 호버 이펙트를 기반으로
각 이미지마다 filter 속성을 적용하였고,
해당 이미지에 커서를 올릴 경우 설명이 나오도록 만들었습니다.
  • blur 효과
  • brightness 효과
  • sepia 효과
  • hue-rotate 효과
  • contrast 효과
  • grayscale 효과
